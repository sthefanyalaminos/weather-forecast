# ⛅ Weather Forecast

[Translate to English]()

Aplicação web de previsão do tempo, desenvolvida com **HTML**, **CSS** e **JavaScript**. O projeto consome dados de uma API de clima em tempo real e exibe as informações de forma simples e intuitiva para o usuário.

## Funcionalidades

- Busca de previsão do tempo por cidade;
- Exibição de temperatura, condição climática e ícone correspondente;
- Interface responsiva, adaptada para desktop e mobile;
- Feedback visual para cidade não encontrada / erros de busca.

## Tecnologias utilizadas

- **HTML5** – Estrutura da aplicação.
- **CSS3** – Estilização e responsividade.
- **JavaScript** – Lógica de busca, manipulação do DOM e consumo da API.
- **OpenWeatherMap** – Fonte dos dados climáticos.

## Objetivo

O projeto foi desenvolvido como prática de fundamentos de front-end (HTML, CSS e JavaScript), com foco em consumo de API e manipulação do DOM.
